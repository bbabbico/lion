# lion

## 현재 까지 구현된것
- 요구사항 기반으로 완성.
- 숨겨진 함수 이벤트가 실행되도 거의 무반응
- FUNCTIONS TO TARGET 으로 실행될 함수 개수 입력 (20이상 입력추천)
- 🔥 BLAST IT 🔥 으로 실행

## Start

### 필요 패키지 설치
`npm install express`

### 크롬 확장프로그램 추가
- chrome-extension 폴더 확장프로그램으로 등록

### 서버 실행
- 크롬/backend 경로에서 node 서버 실행 후 크롬 확장프로그램 이용.
- `node server.js`
