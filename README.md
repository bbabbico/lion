# lion

## 현재 까지 구현된것
- 요구사항 기반으로 완성.
- 여러 사이트들에서 실행 해보니, 생각보다 권한없음 등의 오류가 뜨는 재밌는 함수를 잘 숨겨놔서 그런 이벤트는 잘안뜨는것 같음
-무차별 액션 모드 (최후의 수단) 함수가 너무 깊숙이 숨어있어 도저히 켤 수 없을 때는 타이머 타임아웃 처리가 되는 대신, 화면에 보이는 무작위 버튼이나 입력창을 강제로 클릭합니다! 만약 30개를 실행 요청했다면 중간에 실패하는 함수들이 생기더라도 최소한 30가지의 역동적인 화면 상호작용(모달 팝업, 탭 이동 등)이 무조건 일어나게 됩니다.



## Start

### 필요 패키지 설치
`npm install express`

### 서버 실행
- 크롬/backend 경로에서 node 서버 실행 후 크롬 확장프로그램 이용.
- `node server.js`
