## 🎰 책임 없는 쾌락은 처음부터 짜릿하지 않았다

진짜 도파민은 결과를 모를 때 터진다.  
버튼은 하나. 다음은 아무도 모른다.

왜 유튜브 숏츠는 재밌는데 금방 질리는가?  
결과가 정해져 있어서다. 다음 영상이 나오고, 광고가 나오고, 또 영상이 나온다. 뇌는 안다. 위험하지 않다는 걸.  
위험하지 않으면 도파민은 찔끔 나온다.

도박이 끊기 어려운 건 중독성 때문만이 아니다.  
**잃을 수도 있다**는 사실이 뇌를 깨우기 때문이다.

## 🚀 수동 설치 가이드 (Installation)

> **안내:** 현재 이 확장프로그램은 "모든 웹사이트에 대한 스크립트 실행"이라는 너무나도 무책임하고 강력한 권한을 요구하기 때문에, Chrome 웹 스토어 심사 통과를 기다리는 대신 **가장 도파민 넘치는 방식(수동 설치)**으로 배포합니다.

1. 이 저장소의 코드를 다운로드합니다. (우측 상단 `Code` ➡️ `Download ZIP` 클릭 후 압축 해제)
2. 크롬 브라우저를 열고 주소창에 `chrome://extensions/`를 입력해 접속합니다.
3. 우측 상단에 있는 **'개발자 모드(Developer mode)'** 스위치를 켭니다.
4. 좌측 상단에 생긴 **'압축해제된 확장 프로그램을 로드합니다(Load unpacked)'** 버튼을 클릭합니다.
5. 앞서 압축을 푼 폴더(`manifest.json` 파일이 포함된 chrome-extension 폴더)를 선택합니다.
6. 브라우저 우측 상단 퍼즐 모양(🧩) 아이콘을 눌러 **'도파민 버튼'을 핀(📌) 고정**하면 준비 완료입니다.

## ❓ 어떻게 작동하는가

도파민 버튼은 그 감각을 웹에 꽂는다. 지금 보고 있는 사이트. 그 안에 숨어 있는 수십 개의 이벤트.  
어떤 건 아무 일도 안 일어난다. 어떤 건 페이지가 뒤집어진다. 어떤 건 네가 예상도 못 한 일이 벌어진다.

1. **접속한다.** (아무 사이트나. 익숙한 곳일수록 좋다)
2. **버튼을 누른다.** (팝업을 열고 클릭)
3. **모름.** (무슨 일이 일어날지는 눌러봐야 안다. 그게 전부다)

## 🛠 기술 스택 (How we built the chaos)

단순한 장난 같지만, 뒤에서는 꽤 복잡한 짓을 합니다.

- **Frontend (Chrome Extension)**: Vanilla JS, Manifest V3, Web Audio API (합성된 BGM & 글리치/펑 사운드)
- **Backend (API)**: Node.js, Express, Playwright (Headless Chromium)
- **Deployment**: Leapcell

버튼을 누르면 백엔드 서버가 대상 페이지의 소스코드를 분석해 실행 가능한 모든 함수(`window.xxx`)와 클릭 이벤트를 긁어모읍니다. 확장프로그램은 그 중 하나를 **무작위로 골라 실제 브라우저 환경에서 냅다 실행**합니다.
